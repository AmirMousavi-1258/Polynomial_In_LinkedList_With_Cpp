Polynomial Class with Linked List

  A C++ implementation of polynomials using singly linked lists. Each term (coefficient and exponent) is stored as a node, with terms kept in descending exponent order.

Features 

  Add terms (automatically combines like terms)

  Print polynomials in readable format

  Overloaded + operator for polynomial addition

  Overloaded * operator for polynomial multiplication

  Maintains terms sorted by exponent
